## 📱 3 步开始下载

### 步骤 1: 安装 App
```
Xcode → 选择 iPhone → 点击 Run
```

### 步骤 2: 复制链接
从任意 App 复制视频链接

### 步骤 3: 粘贴并下载
打开 Coto Down → 粘贴 → 点击下载
