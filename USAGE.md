# Coto Down - 使用指南

## ⚠️ 重要说明

### 我是如何下载视频的？

**不是通过解析页面，而是通过拦截网络请求！**

当您粘贴视频链接时：
1. App 在后台打开网页（使用 WKWebView）
2. 浏览器执行 JavaScript 并播放视频
3. App 拦截视频播放器发起的网络请求
4. 获取真实的视频下载链接
5. 用 URLSession 下载视频

**这意味着：**
- ✅ 支持任何有视频播放的网站
- ✅ 不需要针对每个网站写解析器
- ✅ 自动适应网站变化
- ⚠️ 需要加载网页（需要网络）
- ⚠️ 某些加密视频可能无法下载

## ⚠️ 常见问题

### Q: 为什么下载失败？

**可能原因：**
1. **需要登录**
   - 某些视频需要账号登录
   - 当前不支持登录功能

2. **DRM 保护**
   - Netflix, HBO 等使用 DRM
   - 无法下载

3. **页面未完全加载**
   - 网络太慢
   - 超时了

4. **视频未播放**
   - 某些网站需要用户交互
   - 可能需要手动点击播放

**解决方案：**
- 检查网络连接
- 尝试其他视频
- 增加等待时间
- 使用直接链接代替

---

### Q: 为什么这么慢？

**原因：**
- 需要加载完整的网页
- 需要执行 JavaScript
- 需要等待视频缓冲

**优化建议：**
- 使用 WiFi 连接
- 避免高峰时段
- 耐心等待（首次使用 15-30 秒）

---
