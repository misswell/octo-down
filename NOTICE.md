# Notices

## Seal

coto down is inspired by Seal:

- Repository: https://github.com/JunkFood02/Seal
- Summary from Seal README: video/audio downloader for Android, based on
  `yt-dlp`, with playlist download, subtitles, metadata, custom command
  templates, in-app download management, and Material Design 3 UI.

Seal's README warns against using the Seal name for downloader derivatives. This
project uses a separate name: coto down.
