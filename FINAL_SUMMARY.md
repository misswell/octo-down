## 📱 功能特性

### ✅ 完全本地解析
- **无需服务器** - 所有操作在 iPhone 上完成
- **无需配置** - 粘贴链接即可下载
- **完全隐私** - 不发送数据到外部
- **离线工作** - 仅需网络加载网页

### ✅ 智能解析引擎
- **多平台支持** - 自动识别平台
- **智能降级** - 本地解析失败时尝试外部 resolver
- **格式优化** - 自动选择最佳画质
- **批量下载** - 支持多 URL 同时下载

### ✅ 中国平台特殊支持

#### 哔哩哔哩
- ✅ DASH 格式（分离音视频）
- ✅ FLV 格式（合并音视频）
- ✅ H.264/H.265 编码
- ✅ 自动提取封面和时长

#### 抖音
- ✅ RENDER_DATA 提取
- ✅ 自动去除水印（如果可用）
- ✅ 支持短链接和长链接

#### 小红书
- ✅ __INITIAL_STATE__ 提取
- ✅ H.264/H.265 编码
- ✅ 视频笔记下载

## 🔧 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                    Coto Down App                        │
├─────────────────────────────────────────────────────────┤
│  用户输入 URL                                           │
│         ↓                                               │
│  VideoResolverService (统一解析器)                       │
│         ↓                                               │
│  ┌─────────────────────────────────────────────────┐   │
│  │           LocalVideoResolver (本机解析)           │   │
│  │  ┌─────────┬──────────┬──────────┬─────────┐    │   │
│  │  │ YouTube │ Bilibili │ Douyin   │ XiaoHong│    │   │
│  │  │  Vimeo  │ TikTok   │ Twitter  │ Shu     │    │   │
│  │  │  ...    │          │          │         │    │   │
│  │  └─────────┴──────────┴──────────┴─────────┘    │   │
│  └─────────────────────────────────────────────────┘   │
│         ↓                                               │
│  ┌─────────────────────────────────────────────────┐   │
│  │           BackendResolver (外部 resolver)         │   │
│  │           (可选，用于降级)                         │   │
│  └─────────────────────────────────────────────────┘   │
│         ↓                                               │
│  获取直接下载链接                                        │
│         ↓                                               │
│  URLSession 下载                                        │
└─────────────────────────────────────────────────────────┘
```
